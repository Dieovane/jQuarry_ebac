📝 Lista de Tarefas

Como usar:

  • Escreva a tarefa desejada no campo de texto abaixo;
  • Clique no botão 'Adicionar' para colocar a tarefa na lista;
  • Quando a tarefa estiver completa, é só clicar nela para ser riscada da lista!
  • 🌟 Recarregue a página se quiser começar a lista do zero novamente.
